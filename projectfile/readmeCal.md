#-Explanation:


HTML -:
-------
The div with the class container centers the calculator on the page.
The div with the class calculator contains a form with inputs for the display and buttons.
==========================================================================================

CSS -:
------
The body is styled to remove default margins and padding, set the font, and apply box-sizing.
The container is styled to take up the full viewport height and center its content.
The calculator is given a background color, padding, and rounded corners.
The input buttons are styled with no border or outline, fixed height and width, rounded corners, shadow effects, and custom font size and color.
The display input is styled to align text to the right and adjust its size.
===========================================================================================

JavaScript -: (inline with onclick attributes):
-------------
AC Button: Clears the display.
DE Button: Deletes the last character in the display.
Number and Operator Buttons: Appends their value to the display.
Equal Button: Evaluates the expression in the display and updates the display with the result.
